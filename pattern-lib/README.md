# Pattern Library

**Pattern Library** - это ядро для работы с архитектурными паттернами в рамках
расширения языка [Structurizr DSL](https://structurizr.org/dsl).

Библиотека предоставляет базовые абстракции и утилиты, которые позволяют описать архитектурный паттерн
в виде Java класса.

## Сборка и локальная публикация

Для сборки и публикации `debug` версии библиотеки в локальный Maven-репозиторий выполните:

```bash
./gradlew publishDebugPublicationToMavenLocal
```

Для сборки и публикации релизной версии библиотеки в локальной Maven-репозиторий выполните:

```bash
./gradlew publishReleasePublicationToMavenLocal
```

После этого `pattern-lib` будет доступна другим модулям проекта, подключающим её как зависимость.
